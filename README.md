# UnityWebRTCSample
## About
This project is a sample for connecting with [webappsample](https://github.com/masanori840816/webappsample) using Unity WebRTC(https://docs.unity3d.com/Packages/com.unity.webrtc@3.0/manual/index.html).

## Environments
* Unity 6000.2.13f1
* Unity WebRTC 3.0.0-pre.7
